# Mentorship-Capstone-Project

- Mentorship Capstone Project is a full stack website where people can connect with mentors around the world!
# Features
- Password data encryption
- Reset password feature in case a user loses access to their account
- Calculates similarity scores using min-max normalization to match you with a mentor based on your location and more factors
- Utilizes external APIs and a distance algorithm to find a cafe for both parties to meet midway
- Takes your real time location and displays nearby mentors in your area

# Installation (Node.js Installation Required)
- Clone the repository to your local machine using "git clone"
- Run "npm install" to install all necessary libraries
- Add the API keys required for the project in a .env file
- Run npm start
